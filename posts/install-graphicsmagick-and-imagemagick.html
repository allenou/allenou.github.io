# 安装 GraphicsMagick 和 ImageMagick

gm 是 nodejs 的一个图片处理模块，它依赖于 GraphicsMagick 和 ImageMagick，下面是我在 Windows 和 Centos 中安装这两个工具的步骤：

# Windows

Windows 中的安装步骤比较简单，首先到 [GraphicsMagick] 和 ImageMagick 官网下载相应的安装包，然后双击进行默认安装，安装成功后一定要重启电脑。

# Centos

Centos 中的安装步骤则稍微复杂点，这里分开讲解 GraphicsMagick 和 ImageMagick：

# GraphicsMagick

首先找到相应版本的安装包，然后下载：

wget https://sourceforge.net/projects/graphicsmagick/files/graphicsmagick/1.3.33/GraphicsMagick-1.3.33.tar.xz

下载成功后解压：

cd GraphicsMagick-1.3.33
tar -xvf ./GraphicsMagick-1.3.33.tar.xz

解压后再配置：

./configure && make && make install

最后查看是否安装成功：

gm version

这里如果没安装 gcc 则会提示：

configure: error: no acceptable C compiler found in $PATH 
See `config.log' for more details.

解决办法时安装 gcc：

yum install gcc

# ImageMagick

ImageMagick 的安装步骤和 GraphicsMagick 差不多，首先下载安装包：

wget https://www.imagemagick.org/download/ImageMagick.tar.gz

然后解压安装包：

tar -xvf ./ImageMagick.tar.gz

解压后配置：

cd ImageMagick
./configure && make && make install

最后查看是否安装成功：

magick -version
