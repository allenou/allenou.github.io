# 执行上下文

执行上下文（Execution Context）是 ECMAScript 规范中用来描述 JavaScript 代码执行的抽象概念。在 Javascript 中有以下三种执行上下文：

  • 全局执行上下文（Global Execution Context）
  • 函数执行上下文（Function Execution Context）
  • eval 执行上下文

全局执行上下文最先被创建，它是其它执行上下文创建的总入口，它包括变量对象（Variable Object）、作用域和 this 指向，函数执行上下文在函数调用时被创建，它包括活动对象（Action Object）、作用域和 this 指向。

TIP

全局执行上下文只会创建一个，函数执行上下文每次函数调用都会创建。

# 变量对象/活动对象

变量对象和活动对象也无差别，只是因为它们所处执行环境不一样，所以名称也不一样，它们存储的都是当前执行环境中的变量声名和函数声名。

// 全局执行上下文
var sayHello = 'Hello';

function person() {
  //  person 执行上下文
  var first = 'Allen';
  var last = 'AuYeung';

  function firstName() {
      //  firstName 执行上下文
      return first
  }

  function lastName() {
      //  lastName 执行上下文
      return last;
  }
  
  return firstName()+' '+lastName()
}
