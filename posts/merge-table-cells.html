# table 合并单元格

虽然表格布局的时代已经过去，但是并不代表就已经一无是处，只要用对地方仍是一把利器。例如要实现下面的布局 table 就是最好的选择了：

Merokok Malam Kanker Paru Jumlah
YA Tidak
Ya 3 0 3
Tidak 1 3 4
Jumlah 4 3 7

需要用到rowspan（可横跨行数）和colspan（可横跨列数）。

# Step1

<table border="1px" cellspacing="0px">
  <tr>
    <td rowspan="2">Merokok Malam</td>
    <td >Kanker Paru</td>
  </tr>
  <tr>
    <td>YA</td>
  </tr>
</table>
Merokok Malam Kanker Paru
YA

应用了rowspan的单元格独自占一行。

# Step2

<table border="1px" cellspacing="0px">
  <tr>
    <td colspan="2">Kanker Paru</td>
  </tr>
  <tr>
    <td>YA</td>
    <td>Tidak</td>
  </tr>
</table>
Kanker Paru
YA Tidak

应用了colspan的单元格独自占一列。

# Step3

<table border="1px" cellspacing="0px">
  <tr>
    <td rowspan="2">Merokok Malam</td>
    <td colspan="2">Kanker Paru</td>
    <td rowspan="2">Jumlah</td>
  </tr>
  <tr>
    <td>YA</td>
    <td>Tidak</td>
  </tr>
</table>
Merokok Malam Kanker Paru Jumlah
YA Tidak
Ya 3 0 3
Tidak 1 3 4
Jumlah 4 3 7
