# 获取 URL 参数

在 URL 中传参是很常见的跨页面通信方式，而获取 URL 中的参数则是该方式中不可缺少的一步。下面是几种获取 URL 中参数的几种方式：

# Way1:

URL 中参数部分是 ? 号之后的部分，该方法是通过将多次针对性截取到的参数放入对象并返回该对象：

function query(url) {
	var obj = {}
	var q = location.search
	if (q && q.length > 1) {
		var p = q[1].split('&')
		if (p && p.length > 1) {
		    for (var i = 0; i < p.length; i++) {
				var k = p[i].split('=')
				obj[k[0]] = k[1]
			}
		}

	}
	return obj
}
var url = 'https://www.toyou.xyz?name=allen&age=18'
var params = query(url)
parms.allen // allen
params.age // 18

# Way2:

该方法是通过 URLSearchParams 接口接收一个查询 url（即 ? 号及 ? 号之后的部分），直接返回一个参数对象：

var params = new URLSearchParams('?name=allen&age=18')
params.get('name') // allen
params.get('age') //18

URLSearchParams 接口提供了完整的操作 URL 的方法，鉴于本文只讲获取参数，故这里不再拓展，如有兴趣请上 MDN 查看相关介绍。

需要注意的是 URLSearchParams 目前存在一定的兼容性问题，如要使用还是要考虑兼任性问题。

其实完全可以自己整一个有相同 API 的 polyfill,当然也可以使用别人的 polyfill，url-search-params-polyfill 就是别人家的 polyfill。
